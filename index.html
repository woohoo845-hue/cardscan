🔗 Connect Google Sheets

One-time setup — ~3 minutes. After this every card goes straight into your shared team sheet.

1
Open script.google.com → New project → delete any existing code.
2
Tap Copy Script below, paste it in, save (💾).
3
Deploy → New deployment → Web app → Execute as: Me → Access: Anyone → Deploy → authorize → copy the URL.
4
Paste that URL below and tap Save.
